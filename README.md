# Tweet-Analytics
This project focuses on analyzing large-scale social media data to extract meaningful insights and identify behavioral patterns. The study uses a dataset of 1.6 million tweets along with daily website traffic data spanning 5 years to perform advanced analytics using multiple data science techniques.
The project integrates text mining, sentiment analysis, probability modeling, time series analysis, and clustering techniques to understand user behavior and trends across social platforms.

Key Objectives

Analyze tweet content to identify frequently used words and themes

Perform sentiment analysis to classify emotions (positive, negative, fear, joy, etc.)

Study temporal patterns in user activity and sentiment trends

Apply probability distributions (PMF & CDF) to model tweet frequency

Use hierarchical clustering to group similar words and sentiments

Compare tweet activity with website traffic patterns

⚙️ Methodology

Text Mining: Cleaned tweet data by removing stop words, mentions, and noise to extract meaningful words

Sentiment Analysis: Used NLP techniques (e.g., NRC lexicon) to classify emotions in tweets

Clustering: Applied hierarchical clustering to group words based on sentiment similarity

Probability Analysis: Modeled tweet frequency using PMF and CDF distributions

Time Series Analysis: Examined trends across days and over months to identify peak activity periods
